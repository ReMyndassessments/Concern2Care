DIFFERENTIATED LESSON PLAN FOR ABBY P

Lesson:

Linear Equations in Slope-Intercept Form (Modified)

Grade:

9

Duration:

50 minutes (with flexible time allocation)

Focus Student:

Abby P (EAL Intermediate, counseling support needs)

Differentiated Learning Objectives:


1. Identify slope (m) and y-intercept (b) in simplified equations using visual cues
2. Graph a line using step-by-step guided process with physical manipulatives
3. Write basic equations when given slope and y-intercept using sentence frames

Modified Materials:


  • Colored graphing paper (quadrant I only initially)

  • Wikki Stix or string for physical line creation

  • Slope-intercept form graphic organizer (bilingual English/Chinese)

  • Calculator with large display

  • Vocabulary cards with visuals

  • Tablet with graphing app (Desmos or GeoGebra)

  • Modified worksheet with reduced problems and visual supports

Lesson Outline:

1. Warm-Up (8 minutes)


  • Modified problems with visual supports:

  • Graph with points (2,4) and (4,8) pre-plotted with connecting line

  • Equation y = 2x + 5 with color-coding (m=blue, b=green)

  • Teacher uses think-aloud strategy with gestures and simplified language

  • Abby uses vocabulary cards to match terms "slope" and "y-intercept" to visual examples

2. Introduction (12 minutes)


  • Use concrete example: "Phone plan: $10 basic fee + $2 per GB"

  • Physical demonstration with stacking blocks: base (b) and incremental adds (m)

  • Graphic organizer with fill-in-the-blank: y = __ x + __

  • Bilingual vocabulary support: slope (坡度), y-intercept (y截距)

  • Chunked information: Teach m and b separately with practice after each

3. Guided Practice (15 minutes)


  • Example 1: y = 2x + 1 using tactile materials

  • Place Wikki Stix at (0,1) on coordinate plane

  • Use "rise/run" cards: ↑2 →1 with physical movement

  • Tablet graphing app to verify results

  • Example 2: y = -½x + 4 with emphasis on negative slope visualization

  • Abby works with partner for think-pair-share after each step

4. Independent Practice (12 minutes)


  • Modified worksheet with 4 problems instead of multiple sections:

1. Circle slope and underline y-intercept in y = 3x - 2
2. Graph y = x + 2 using provided coordinate plane with starting point marked
3. Complete equation: y = __ x + __ when given m=2, b=1
4. Match graph to equation from two choices
  • Teacher provides immediate feedback and corrective instruction

5. Closing & Assessment (3 minutes)


  • Modified exit ticket:

  • Multiple choice: "What is the y-intercept in y = 4x + 3?" with options

  • Partial graph completion: Continue the line from given starting point

  • Verbal explanation option: "Tell me how you would graph y = 2x + 1"

Homework Adaptation:


  • 4 problems total with visual supports

  • Option to complete digitally or on paper

  • Parent guide sheet in Chinese explaining the concepts

COMPREHENSIVE DIFFERENTIATION STRATEGIES

Research-Based Content Modifications


  • Sheltered Instruction Observation Protocol (SIOP) techniques: Content and language objectives explicitly stated, vocabulary emphasized with visual supports (Echevarria, Vogt, & Short, 2017)

  • CRA Framework: Concrete (manipulatives), Representational (graphs), Abstract (equations) progression for mathematical understanding (Witzel, Mercer, & Miller, 2003)

  • Cognitive load reduction: Chunking information, eliminating extraneous processing through worked examples and guided practice (Sweller, 2010)

Process Adaptations


  • Think-aloud modeling: Teacher demonstrates problem-solving process verbally while solving

  • Gradual release: I do (teacher models), We do (guided practice), You do (independent practice) with extended guided phase

  • Multimodal input: Visual, auditory, kinesthetic, and tactile learning opportunities integrated

  • Wait time: Minimum 10 seconds for processing questions and responses (Rowe, 1986)

Assessment Modifications


  • Alternative response modes: Verbal responses, pointing, matching, selection rather than exclusively written

  • Portfolio assessment: Collection of work samples over time to show growth rather than single-point assessment

  • Dynamic assessment: Test-teach-retest model to measure learning potential rather than static ability (Lidz, 1991)

Environmental Supports


  • Strategic seating: Near teacher for proximity support, away from distractions

  • Visual schedule: Clear timeline of lesson activities with visual cues

  • Break options: Pre-arranged signal for 2-minute break if overwhelmed

  • Reduced auditory distractions: Noise-reducing headphones available

Technology Integration


  • Graphing calculator apps: Desmos or GeoGebra for visual representation

  • Translation tools: Google Translate or similar for vocabulary support

  • Speech-to-text: For written responses if needed

  • Digital graphic organizers: Modify and save for future reference

Progress Monitoring System

  • Behavioral observation: Engagement and participation metrics using interval recording

Collaboration Plan

Implementation Timeline
